## Chosen workflow
Drafting customer support email responses for a small online store.

## User
The user is a customer support assistant who handles common customer emails.

## Input
The system receives a customer email describing an order-related issue, such as delayed shipping, a damaged product, the wrong item, or a return request.

## Output
The system should produce a polite and professional first-draft email response. The response should acknowledge the customer’s issue, provide a helpful next step, and maintain a consistent support tone.

## Why this task is valuable
This workflow is repetitive and depends heavily on written communication. Support staff often spend time replying to similar issues. A GenAI system can help by creating a fast first draft for common cases. However, some cases, especially refunds, policy exceptions, or angry customers, may still need human review.
